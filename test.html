Arc - The curved scale on the sextant used to measure angles.

Index Arm - The adjustable arm on the sextant that moves along the arc, used to measure the angle between the horizon and a celestial body.

Index Mirror - A small mirror attached to the index arm, which reflects the celestial object to allow the observer to take readings.

Horizon – The reference line between sea and sky.

Altitude – Angle between a celestial body and the horizon.

Arc – Curved scale marked in degrees.

Index Mirror – Mirror attached to the index arm.

Horizon Mirror Mirror used to align the object with the horizon.

Telescope Small scope used to view the horizon and celestial body.

Vernier Scale Fine scale for precise angle readings.

Index Error Error caused by mirror misalignment.

Celestial Body Sun, moon, planet, or star being observed.

Zenith Point directly overhead (90° altitude).

Celestial Navigation Navigation using stars and the sun.

Latitude North–south position found using sextant observations.
